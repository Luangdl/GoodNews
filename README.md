# GoodNews
Estudos sobre consumo de API e modelagem de dados conformando com Codable, utlizando recursos URLSession.shared e JSONDecoder.

Aplicativo construido utlizando a arquitetura MVC, onde foi realizada a injeção de dependência no SceneDelegate deixando o responsável por realizar a busca dos objetos entre a controller e a model, com menor acoplamento, facilitando assim alterações futuras na model e possíveis testes, caso necessário.
